Pinterest Addon Basic

The Pinterest Addon gives you the ability to add a beautiful “Pin It” button on top of each image in your WordPress slider. It is super simple to use.

First, there is an option to enable the “Pin It” button for each slider, and then you can choose the positioning of the button – top left, top right, bottom left, and bottom right.

Beyond this there are, as always, hooks and filters so that you can modify data as needed. The Addon will use the image title as the description and will attempt to link to the current post. If there is no current post (such as on an archive page), it will link to your home page instead.

As with Addons, a developer or lifetime license is required to access them.
